// event bubling is a concept in JavaScript where an event triggered on a child element propagates up to its parent elements. For example, if you have a button inside a div and you click the button, the click event will first trigger on the button and then bubble up to the div, allowing you to handle the event at different levels of the DOM hierarchy. This can be useful for managing events efficiently and avoiding unnecessary event listeners on multiple child elements.
